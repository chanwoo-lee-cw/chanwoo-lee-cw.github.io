# github.io 를 이용한 포트폴리오

## 목적
포트폴리오를 GitHub.io를 이용해 작성한다.

## 요구사항
- 랜딩 페이지로 작성한다
- 참고 데이터를 이용해서 포트폴리오를 작성한다.
- frontend-design 플러그인을 활용한다.
- gitHub.io를 통해 접근이 가능하게 한다.
- 데이터만 사용하면 되고, 포멧에는 구애될 필요가 없다.
- 이력서의 대소문자 정리가 필요
- 밝은 분위기의 테마로 해줘
- 포트폴리오의 프로젝트 정보는 클릭해서 옮겨갈 수 있도록 한다
- 프로젝트 파일은 원본에 있는 이미지 파일같은 것도 전부 표기할 수 있도록 한다
  - 프로젝트 하위 데이터를 보고 데이터도 넣어줘
- mermaid로 되어 있는 다이어 그램들은 이미지 파일로 변환해서 넣어줘
