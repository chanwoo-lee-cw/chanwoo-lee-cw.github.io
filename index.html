01 About Me

안정적인 시스템,
지속 가능한 코드

Kotlin / Spring Boot 기반으로 상품 카탈로그 시스템과 오더 관리 플랫폼을 개발해온 백엔드 개발자입니다. 레거시 Python Flask 시스템을 Kotlin 기반으로 점진적으로 전환하는 마이그레이션을 주도하며, 복잡한 상속·리플렉션 구조를 명시적인 MVC 계층으로 개선한 경험이 있습니다.

단순히 기능을 구현하는 데서 그치지 않고, DB 정규화, Null-safe 설계, Soft Delete 도입 등 데이터 일관성과 유지보수성을 함께 고려합니다.

Apache Kafka, Redis 분산락, Spring Batch를 활용한 비동기 처리와 대용량 데이터 처리 경험이 있으며, AWS Lambda와 CloudFront를 활용한 이미지 처리 최적화(80% 이상 단축) 등 인프라 레벨까지 고려한 시스템 설계를 지향합니다.
